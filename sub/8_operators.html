Operators

비교연산자: true, false

논리연산자: Logical operators

연산자: OPERATORS
 ◾ 할당연산자: 변수값을 할당 =
    let x = 10;
 ◾ 덧셈연산자: 값을 더함 +
    let x = 5; let y = 2; let z = x + y
 ◾ 곱셈연산자: 값을 곱함 *
    let x = 5; let y = 2; let z = x * y
 ◾ 비교연산자: 값을 비교 >
    let x = 5; let y = 2; let z = x > y


자바스크립트 연산자 유형
 ◾ 산술연산자: [+, -, *, **, /, %, ++, --]
 ◾ 할당연산자: [=, +=, -=, *=, /=, &=]
 ◾ 비교연산자: [==, ===, !=, !==, >, <, >=, <=]
 ◾ 문자열연산자: [+, +=]
 ◾ 논리연산자: [&&, ||, !]
 ◾ 비트연산자: [&, ||, ^, ~, <<, >>]
 ◾ 삼항연산자: if문, 조건이 참일때 : 거짓일때


논리연산자: Logical operators
 ◾ &&: 조건 두개가 일치
 ◾ ||: 둘중에 하나라도 일치
 ◾ !: 거짓이 진실, 진실이 거짓
