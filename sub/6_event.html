html 이벤트

EVENT
HTML 이벤트는 브라우저가 수행하는 일이거나 사용자가 수행하는 일
 ◾ 페이지 로딩 완료
 ◾ 입력 필드 변경
 ◾ 버튼 클릭


HTML EVENT
 ◾ onchange: 요소가 변경될때
 ◾ onclick: 엘리먼트 클릭할때
 ◾ onmouseover: 마우스 이동할때(hover시)
 ◾ onmouseout: 마우스 멀리 이동할때
 ◾ onkeydown: 키보드 누를때
 ◾ onload: 페이지 로딩 완료할때
